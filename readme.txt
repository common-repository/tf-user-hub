TF User Hub displays the information of a themeforest author/user including the newest products as a shortcode and widget.

== Description ==

TF User Hub displays Themeforest Author's username, country, sales, followers and all public information including the thumbnail image.
The badges of an Author/User have earned so far can be displayed in here as well.
It's built using Envato's latest Api.
According to the user's requirement the number of latest Items can be displayed as well.



== Installation ==

1. Download "TF User Hub" plugin
2. Simply go to the Plugins page, then click on Add new and select the plugin's .zip file which is "tf-user-hub.zip".
3. Alternatively you can extract the contents of the zip file directly to your wp-content/plugins/ folder
4. Finally, activate the plugin.

== Frequently Asked Questions ==

= Is there any shortcode to use it in a page or post? =

Yes. Put the shortcode [tf_user username="" item_count=""] with these additional parameters.
"username" is your Themeforest username. e.g: themepicasso

"item_count" is the number of latest items you want to display in the post or page.
e.g: 4

= Can I use it in my sidebar widgt? =

Yes. You can. Simply Drag and Drop the widget "Themeforest User Hub" in your desired Sidebar Area and fill up the config like you would do in your shortcode.
Done :)
